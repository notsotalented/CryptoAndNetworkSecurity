# CryptoAndNetworkSecurity
 Just a basic webpage for the assignment

How To Use:
1) Move the folder to XAMPP installation (e.g C:\\xampp) and rename the folder to "htdocs".
2) Open XAMPP, start both Apache and MySQL.
3) Access MyPHPAdmin via XAMPP->mySQL->Admin
4) Import the database (login.sql) via Import->Choose "login.sql" in the folder "database_here"
